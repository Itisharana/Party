# 🎉 PartyPoper – Event Platform UI

A responsive and modern **event platform UI** where users can explore, host, and join parties.
This project focuses on clean design, card layouts, and responsive structure using Bootstrap.

## 🚀 Features

* Responsive layout for all devices
* Clean navigation bar with search functionality
* Interactive cards for:

  * Host Party
  * Join Party
* Overlay text design on images
* Simple and user-friendly UI

---

## 🛠️ Technologies Used

* **HTML5** – Page structure
* **CSS3** – Styling and custom UI
* **Bootstrap 5** – Layout and responsiveness
